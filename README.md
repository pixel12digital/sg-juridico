# Tema SG Juridico

Tema WordPress personalizado para SG Juridico.

## Características

- **Carrossel Alimentado pelo Painel**: Carrossel isolado que pode ser configurado através do painel administrativo
- **Integração WooCommerce**: Suporte completo para e-commerce
- **Layout Responsivo**: Design adaptável para diferentes dispositivos
- **Customização Avançada**: Painel de configurações dedicado

## Estrutura

O tema está localizado em `theme-only/sg-juridico/`.

### Componentes Principais

- `front-page.php`: Template da página inicial
- `template-parts/home-carousel.php`: Componente isolado do carrossel
- `functions.php`: Funcionalidades e configurações do tema
- `header.php` / `footer.php`: Cabeçalho e rodapé
- Páginas customizadas: `page-sobre.php`, `page-contato.php`

### Arquivos de Estilo

- `style.css`: Estilo principal do tema
- `css/palette.css`: Paleta de cores
- `css/contact.css`: Estilos da página de contato
- `css/my-account.css`: Estilos da conta do usuário

### Scripts JavaScript

- `js/navigation.js`: Navegação e menu
- `js/calendario.js`: Calendário de eventos
- `js/eventos-expand.js`: Funcionalidade de expansão de eventos
- `js/contact-form.js`: Formulário de contato

## Instalação

1. Fazer upload da pasta `theme-only/sg-juridico/` para `wp-content/themes/` no seu WordPress
2. Ativar o tema no painel administrativo
3. Configurar as opções do tema através do menu "SG Juridico"

## Carrossel

O carrossel da página inicial está totalmente isolado e pode ser configurado através do painel administrativo. Os tipos de slides suportados são:

- Texto com título, subtítulo, descrição e botão CTA
- Categorias de produtos
- Produtos individuais
- Imagens com overlay de texto

## Desenvolvimento

Este tema foi desenvolvido seguindo as práticas recomendadas do WordPress e inclui:

- Código limpo e bem documentado
- Hooks e filtros do WordPress
- Suporte a temas filho
- Validação e escape de dados
